# Style-Store

Style-Store is a full-stack web application built for exploring and managing fashion products online. It allows users to browse clothing items, save favorites, manage wishlists, place orders, and interact with products through reviews and ratings. The application is designed to provide a smooth shopping experience with both user-facing features and backend management support.

## Features
- User registration and login
- Browse and search clothing items
- Add items to wishlist and favorites
- Write comments, reviews, and ratings
- Create and manage orders
- Payment support
- Notifications for discounts and updates
- Upload and manage product photos

## Tech Stack
- ASP.NET Core
- Angular
- SQL Server
- Redis
- Docker

## Project Structure
- `Frontend` for the user interface
- `Backend` for APIs and business logic
- Database for storing product, user, and order data
- Caching support for better performance

## Run the Project
1. Clone the repository
2. Update the required configuration in `appsettings.json`
3. Build and run the application
4. Start the frontend and backend services

## Purpose
This project was built to create a modern fashion shopping platform with essential e-commerce features, user interaction, and scalable application design.
